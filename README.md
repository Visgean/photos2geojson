# photos2geojson
Searches recursively for photos and generates geojson map


### Install: 

```
$ pip3 install photos2geojson --user
```


### Usage:

```
$ photos2geojson ~/Dropbox/ -g
```

you can also skip parameter -g and generate normal geojson file on your drive.
